## What We Build

### 🔧 [intrig](https://intrig.dev) — Open-Source API Tooling

intrig is a TypeScript code generator and SDK builder for OpenAPI specifications. Drop in your OpenAPI spec and get fully-typed API clients, documentation, and integration utilities — no boilerplate.

- **Zero runtime overhead** — generates clean TypeScript, not a wrapper library
- **Framework-agnostic** — works with any Node.js or React project
- **OpenAPI 3.x support** with ongoing feature expansion

→ [`intrigsoft/intrig-core`](https://github.com/intrigsoft/intrig-core)

---

### 🤖 [Dioschub](https://dioschub.com) — Enterprise AI Orchestration

Dioschub connects enterprise AI to legacy systems without exposing sensitive credentials or data. Built around a **zero-trust architecture**, it lets organizations safely deploy AI agents across existing infrastructure.

Key capabilities:
- **Credential Blind Pattern** — AI agents never see or store user credentials
- **BYOA** — Bring Your Own Agent; integrate any LLM or agent framework
- **MCP Proxy** — Model Context Protocol orchestration layer
- **RBAC + Audit** — Role-based access with full traceability
- **Page Intelligence** — Document and knowledge retrieval for legacy content

→ [dioschub.com](https://dioschub.com)
